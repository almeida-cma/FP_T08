📌 1. Identificadores (Nomes de Variáveis)

Regras para nomear variáveis corretamente

✅ Exemplo CORRETO 
programa
{
    funcao inicio()
    {
        // ✅ Identificadores VÁLIDOS
        cadeia nome_Cliente = "João"
        inteiro data_Emissao = 2024
        real valor_1_a_100 = 75.5
        
        escreva("Nome: ", nome_Cliente, "\n")
        escreva("Data: ", data_Emissao, "\n")
        escreva("Valor: ", valor_1_a_100, "\n")
    }
}
❌ Exemplo INCORRETO 
programa
{
    funcao inicio()
    {
        // ❌ Identificadores INVÁLIDOS (NÃO funcionam!)
        // cadeia nome do cliente = "João"   // tem espaço!
        // inteiro data_Emissão = 2024       // tem acento!
        // cadeia MensagemAviso!!! = "ok"    // tem caracteres especiais!
        // inteiro 2aSemana = 5              // começa com número!
        
        escreva("Use apenas letras, números e _\n")
        escreva("Não use espaços, acentos ou caracteres especiais\n")
    }
}

📖 Regras para identificadores: Começar com letra, usar apenas letras/números/underscore (_), sem espaços, sem acentos, sem caracteres especiais.

📌 2. Indentação de Código

Organizando o código com recuos

✅ Com Indentação (Código Legível) 
programa
{
    funcao inicio()
    {
        inteiro i = 0
        
        para (i = 1; i <= 5; i++)
        {
            se (i % 2 == 0)
            {
                escreva(i, " é par\n")
            }
            senao
            {
                escreva(i, " é ímpar\n")
            }
        }
    }
}
❌ Sem Indentação (Código Confuso) 
programa
{
funcao inicio()
{
inteiro i = 0
para (i = 1; i <= 5; i++)
{
se (i % 2 == 0)
{
escreva(i, " é par\n")
}
senao
{
escreva(i, " é ímpar\n")
}
}
}
}

📖 Indentação: Recuar linhas de código para mostrar hierarquia e facilitar a leitura. Código indentado é mais fácil de entender e dar manutenção.

📌 3. Comentários no Código

Documentando o programa

✅ Código com Comentários 
programa
{
    // ==========================================
    // PROGRAMA: Calculadora de Média
    // AUTOR: Aluno
    // DATA: 09/04/2026
    // ==========================================
    
    funcao inicio()
    {
        real nota1, nota2, media  // Declara variáveis
        
        // Entrada de dados
        escreva("Digite a primeira nota: ")
        leia(nota1)
        
        escreva("Digite a segunda nota: ")
        leia(nota2)
        
        // Processamento
        media = (nota1 + nota2) / 2
        
        // Saída do resultado
        escreva("Média: ", media, "\n")
    }
}

📖 Comentários: Explicam o que o código faz. Ajudam outros programadores (e você mesmo no futuro) a entender o programa.

📌 4. Diminuindo Linhas de Código

"Menos é mais" - Código conciso e eficiente

❌ Código MUITO extenso 
programa
{
    funcao inicio()
    {
        inteiro a, b
        
        escreva("Digite A: ")
        leia(a)
        
        escreva("Digite B: ")
        leia(b)
        
        escreva("Soma: ", a + b, "\n")
    }
}
✅ Código MAIS CONCISO 
programa
{
    funcao inicio()
    {
        inteiro a, b
        
        escreva("Digite A e B: ")
        leia(a, b)  // Leitura múltipla em uma linha!
        
        escreva("Soma: ", a + b, "\n")
    }
}

📖 Diminuir linhas: Código mais limpo e direto. Use leia(a, b) para ler múltiplas variáveis. Menos linhas = menos chance de erro.

📌 5. Validação de Dados

Protegendo o programa contra entradas inválidas

✅ Com Validação de Dados 
programa
{
    funcao inicio()
    {
        inteiro idade
        
        faca
        {
            escreva("Digite sua idade (1 a 120): ")
            leia(idade)
            
            se (idade < 1 ou idade > 120)
            {
                escreva("❌ Idade inválida! Tente novamente.\n")
            }
        } enquanto (idade < 1 ou idade > 120)
        
        escreva("✅ Idade válida: ", idade, " anos\n")
    }
}

💡 Dica: Sempre valide entradas do usuário! Verifique se o valor está dentro do esperado, se não está vazio, se é do tipo correto.

📌 6. Programa Completo com TODAS as Boas Práticas

Unindo indentação, comentários, validação e código limpo

🏆 Programa Exemplar 
programa
{
    // ==========================================
    // PROGRAMA: Sistema de Validação de Idade
    // OBJETIVO: Ler idade do usuário com validação
    // ==========================================
    
    funcao inicio()
    {
        inteiro idade_usuario
        
        escreva("================================\n")
        escreva("   SISTEMA DE VALIDAÇÃO\n")
        escreva("================================\n\n")
        
        // Validação da idade
        faca
        {
            escreva("Digite sua idade (1 a 120): ")
            leia(idade_usuario)
            
            se (idade_usuario < 1 ou idade_usuario > 120)
            {
                escreva("❌ Idade inválida! Digite entre 1 e 120.\n\n")
            }
        } enquanto (idade_usuario < 1 ou idade_usuario > 120)
        
        // Exibe resultado
        escreva("\n✅ Idade registrada: ", idade_usuario, " anos\n")
        
        // Verifica maioridade
        se (idade_usuario >= 18)
        {
            escreva("🔞 Usuário é MAIOR de idade\n")
        }
        senao
        {
            escreva("🧒 Usuário é MENOR de idade\n")
        }
    }
}
